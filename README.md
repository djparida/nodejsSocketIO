# nodejsSocketIO

## Guide to Run the app
  
  npm install
  
  
  nodemon server.js
  
  
## To connect the Node SocketIO server with Python

  cd PythonClient
  
  create Virtualenvironment (python -m venv venv)
  
  To activate - venv\Scripts\activate 
                
                source venv/bin/activate

  pip install socketIO-client
  
  Python client.py
